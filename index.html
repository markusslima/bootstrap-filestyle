Description

The filestyle-bootstrap.js is a plugin for jquery-based component library could Bootstrap, used to style the file fields of the forms.

This plugin has an interesting way to present a form for sending files so attractive, while being simple and efficient.

Requirements

Bootstrap

jQuery

Installation

Include the file bootstrap-filestyle.js your html code.

<script type="text/javascript" src="js/bootstrap-filestyle.js"> </script>

Usage

Basic usage:

$(":file").filestyle()


Options usage:

$(":file").filestyle({
    textField: "My files",
    icon: true
})

Options

buttonText String Customization of button text.
textField Bool Show and hide text field.
classButton String Use of classes Bootstrap for button.
classText String Use of classes Bootstrap for text field.
icon Bool Use icon.
classIcon String Use of classes Bootstrap for icons.

Methods

clear String Clears a set filestyle.
